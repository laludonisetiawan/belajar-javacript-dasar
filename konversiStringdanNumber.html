Konversi String dan Number

  • Saat membuat aplikasi, kadang kita input dari pengguna selalu dalam bentuk String.
  • Sedangkan kita ingin mengelola datanya dalam bentuk Number.
  • Maka sangat disarankan untuk melakukan konversi tipe data.

output saat kode javascript di jalankan:

  • masalah tanpa konversi

    saat kita ingin menambahkan niali dari variable 1 dan 2 dengan tipe data yang berbeda maka hasilnya akan menampilkan nilai terakhir sebelumnya, karena javascript akan mendeteksi bahwa kedua variable tersebut memiliki tipedata yang berbeda misal variable1 tipedatanya String sedangkan variable tipe datadanya integer.

output kode javscript tanpa konversi:

misalnya kita ingin mengkonversi tipe data String "1" dengan tipe data Integer 1
maka hasilnya akan menampilkan nilai terakhir sebelum di konversi yaitu string 1 dan integer 1.

Untuk itu kita harus tau cara Agar tipedata string bisa di konversi ke dalam Number.

  1. tanpa konversi string "1" to Number 1.
  2. konversi string "1" to number 1 (bilangan bulat.)
  3. konversi string "1.34" to number 1 (bilangan pecahan)
  4. koversi string "4.56" to number 5 (bilangan bulat atau pecahan.)
  5. koversi number 1 to string "1.7".
